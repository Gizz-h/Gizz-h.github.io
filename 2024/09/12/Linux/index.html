Linux

Linux的应用场景：
Linux下的开发项目—javeEE，大数据，PythonPHP，c，c++，go
Linux的应用领域：
1。服务区领域：linux在服务器领域是最强的，稳定高效
2。嵌入式领域：内核很小（几百kb）可以对软件剪裁等 应用：机顶盒，数字电视，手机，智能家居
Linux的介绍：一个稳定免费开源安全处理多并发的操作系统
Linux主要的发行版：ubuntu redhat centos fedpra suse debain

网络连接的三种模式：

1.桥接模式：虚拟系统可以和外部系统通讯，但是容易造成ip冲突
2.NAT模式：网络地址转换模式，使用一个代理的地址，虚拟系统可以和外部系统通讯，不会造成ip冲突。
3.主机模式：完全独立的系统

虚拟机克隆：

1.直接复制粘贴配置好的虚拟机
2.使用vmware的克隆系统，需要先关闭待克隆的linux

虚拟机快照：

类似于存档，既可以返回之前的快照，返回之前的状态，需要存档时拍摄快照即可虚拟机的迁移和删除：迁移直接剪切，删除就是直接删除或者用vm删除

登录shell

什么是登录 Shell？

登录 shell 就是这样：当你直接登录到 Linux 机器时启动的 shell。当您从虚拟控制台或通过 SSH 登录系统时，启动的 shell 是登录 shell。

相比之下，如果您启动终端窗口，则该 shell 会话通常不是登录 shell。当您通过在命令行中键入 shell 的名称来启动子 shell 时也是如此。这个 shell 只是一个交互式 shell。

两者之间的主要区别在于 shelll 的行为。shell 通常只会在启动时读取某些文件。只有在作为登录 shell 调用时，Bash 才会读取您的主目录中的/etc/profile、.bash_profile、.bash_login和.profile 文件。

简单来讲shell就是将我们的指令进行翻译，使得linux能够识别，起到一个翻译官的作用

管道符号

“|”是Linux管道命令操作符，简称管道符。使用此管道符“|”可以将两个命令分隔开，“|”左边命令的输出就会作为“|”右边命令的输入，此命令可连续使用，第一个命令的输出会作为第二个命令的输入，第二个命令的输出又会作为第三个命令的输入，依此类推。

lslh
